               Blog website

This project is a personal diary website where you can compose your posts and see those posts on your home page. 
Also you can add, update, delete and save your posts the way you want.
 It is done my using Node.js,EJS,  JSON file (is used to store our data), jQuery and various packages like express, body-parser, lodash and others. CRUD operations are used as well by creating new file and is exported to our 
main Javascript file by using module.exports.
